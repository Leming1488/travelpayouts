Быстрый старт Виджета

Шаг 1. Подключите Виджет

Для этого добавьте в заголовок HEAD HTML-страницы строку javascript кода. Это загрузит в браузер JavaScript-файл. 


                <head>
                ...
                <script>(function() { var script = document.createElement('script');
                script.async = true;
                script.src = 'https://travelpayoutsjs-leming1488.c9users.io/widget.js';
                var entry = document.getElementsByTagName('script')[0];
                entry.parentNode.insertBefore(script, entry);})();</script>
                ...
                </head>

Шаг 2. Создайте контейнер

Создайте HTML-элемент блочного типа ( например DIV ) c классoм "js-travelpayoutsjs-widget" ненулевого размера, в котором будет размещен виджет. 

<div class="js-travelpayoutsjs-widget"></div>

Шаг 3. Кастомизация Виджета

Для изменения опций по умолчанию добавьте дополнительные атрибуты в созданый контейнер. 

<div data-size="l" data-title-text="hello" data-background-color="#4A90E2" data-button-color="#F5A623"  class="js-travelpayoutsjs-widget"></div>
Атрибут Пример Описание
Locale data-locale="ru" Набор параметров, специфичных для языка и/или страны.
Например "ru" русский.
(По умолчанию локаль для браузера).
Size data-size="l" Размерная сетка ширины виджета: "xs", "s", "m", "l".
(По умолчанию автоматически задется от родительского контейнера.
Button color data-button-color="#F5A623" Цвет кнопки отправки. Формат цвета Hex Color.
Background color data-background-color="#4A90E2" Цвет фона виджета. Формат цвета Hex Color.
Button text data-button-text="submit" Текст кнопки.
Title text data-title-text="Where does it come from? Why do we use it?" Текст заголовка.
Content text data-paragrph-text="It is a long established fact that a reader will be distracted by the readable content of a page when looking at its layout" Текст основного блока.
Placeholder text data-placeholder1-text="Depart date"
data-placeholder2-text="Return date"		 Тект полей формы.
